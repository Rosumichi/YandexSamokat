# Автотесты для API Яндекс.Самокат

Проект содержит автоматизированный сценарий проверки получения данных о созданном заказе по его трек-номеру в сервисе Яндекс.Самокат.

## Инструкция по запуску тестов

### Предусловия
1. На компьютере должен быть установлен Python версии 3.x.
2. Установлены зависимости проекта (библиотеки `requests` и `pytest`).

### Установка зависимостей
Запустите команду в терминале для установки необходимых пакетов:
```bash
pip install requests pytest
